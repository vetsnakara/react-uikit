При необходимости, у ячейки таблицы может быть указанна минимальная ширина.

Условия: тег - TD, свойство - min-width, единицы измерения - px, таблица - не .table_balanced

Указать минимальную ширину можно у td в любой tr, или даже во всех tr одновременно.

  • .table__wrapper - класс обёртки таблицы;
  • .table__wrapper_noborder - обёртка таблицы без внешней рамки;
  • .table__wrapper_relative - для компонента tree-table, когда таблица заполнена чекбоксами;
  • .table - класс таблицы;
  • .table__header - класс хэдера для thead;
  • .table__head - класс для строки thead;
  • .table__title - класс для ячейки thead;
  • .table__body - класс хэдера для tbody;
  • .table__item - класс для tbody;
  • .table__cell - класс для ячейки tbody;
  • .table__footer - класс строки tfoot. Строки и ячейки у него как у tbody.
