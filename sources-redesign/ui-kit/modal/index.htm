В дизайн-системе есть два базовых типа окон - модальное и диалоговое. Для диалогового требуется добавить окну модификатор .modal_dialog

  • .modal - класс обычного модального окна с разделителями;
  • .modal__content - класс контента модального окна;
  • .modal__header - класс хэдера модального окна, включает заголовок, иконки и управляющие элементы;
  • .modal__title - класс заголовка модалки;
  • .modal__close - класс иконки закрытия модалки;
  • .modal__body - класс основного контента модалки;
  • .modal__footer - класс футера модалки. Нижние управляющие кнопки (сохранить, отмена пр.) помещаются в него. В диалоговом модальном окне должен выводиться хотя бы пустым для корректного вида.
