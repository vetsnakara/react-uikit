Элемент .pagination__item может быть оформлен тэгом a или button по необходимости

  • .pagination - класс контейнера пагинации;
  • .pagination__item - класс элемента пагинации;
  • .pagination__item_previous - класс кнопки назад (предыдущий пункт);
  • .pagination__item_active - класс активного пункта;
  • .pagination__kebab - класс многоточия;
  • .pagination__item_next - класс кнопки вперёд (следующий пункт).
