  • .sidebar-menu - основной класс меню;
  • .sidebar-menu__preview - кнопка для мобильной версии, раскрывающая выпадающий список меню, скрыта на десктопе;
  • .sidebar-menu__list - список ссылок меню;
  • .sidebar-menu__item - ссылка.
