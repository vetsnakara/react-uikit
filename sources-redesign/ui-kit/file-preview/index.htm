Важно соблюдать порядок и вложенность div в превью.

  • .file-preview - основной класс блока превью;
  • .file-preview_extended - модификатор для превью с возможностью скачивания файла;
  • .file-preview__type-img - класс изображения типа файла;
  • .file-preview__wrapper - класс обёртки для названия и типа файла;
  • .file-preview__name - класс название файла;
  • .file-preview__data - класс информации о файле;
  • .file-preview__actions - класс блока исполняемых действий с файлом (удалить, скачать и т.д.);
  • .file-preview__action - класс исполняемого действия файла.

{{ src }}

{{ info }}
- информация о файле (если её нет просто не вставлять этот элемент).
