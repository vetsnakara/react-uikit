Блок .upload__info следует размещать в самом конце label , чтобы корректно работали интерактивные элементы. Визуальны порядок элементов задаётся стилем.

  • .upload - основной класс блока загрузчика;
  • .upload__control - класс управляющего элемента загрузчика;
  • .upload__label - класс кнопки загрузчика;
  • .upload__title - класс заголовка загрузчика;
  • .upload__description - класс информационного текста под заголовком загрузчика;
  • .upload__info - класс блока превью загруженных файлов.
{{ label }}
- текст кнопки загрузчика;
{{ title }}
- текст заголовка;
{{ description }}
- информационный текст под заголовком;
{{ filePreviewComponent }}
- компонент превью файла.
.upload__error - класс ошибки загрузчика;
