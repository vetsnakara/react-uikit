Компонент-оболочка для конструирования древовидных структур

  • .tree-nested - класс оболочки, обнулённый по стилям список
  • .tree-nested_margin-20 - вложенный список с отступом 20 пикселей
  • .tree-nested_margin-16 - вложенный список с отступом 16 пикселей
  • .tree-nested__collapse - класс коллапса, вкладывается в item
  • .tree-nested__collapse_active - модификатор открытого коллапса
  • .tree-nested__collapse_selected - модификатор выбранного коллапса
  • .tree-nested__collapse-text - текст коллапса
  • .tree-nested__collapse-dropdown - управляющие элементы коллапса
  • .tree-nested__collapse-last - последний, не разворачивающийся объект коллапса
