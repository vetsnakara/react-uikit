Не применять tooltip на svg, использовать обертку. Если элемент кликабельный, то в качестве обертки использовать тэг button. Если не кликабельный, то можно в качестве обертки использовать другой тэг.

  • data-tooltip - вызов тултипа.
  • data-placement - положение тултипа. Если не указывать, то auto.
